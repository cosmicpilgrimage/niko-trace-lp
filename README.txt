虹鼓 niko ランディングページ ― 公開手順（GitHub Pages）

1. GitHub で新しいリポジトリを作る（例: niko-trace-lp）。
2. この index.html と images フォルダを、リポジトリの一番上（ルート）にアップロードする。
   - GitHub サイト上で「Add file ▸ Upload files」にドラッグ＆ドロップでOK。
   - index.html と images フォルダの構成（位置関係）は変えないこと。
3. リポジトリの Settings ▸ Pages ▸ Build and deployment で
   Source = Deploy from a branch、Branch = main / (root) を選んで Save。
4. 数分後、https://<ユーザー名>.github.io/<リポジトリ名>/ で公開されます。

※ファイル名（images/ 内）は変更しないでください。index.html が同じ名前で読み込んでいます。
